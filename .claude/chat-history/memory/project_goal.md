Build a technical portfolio / landing page at frah90.github.io (GitHub Pages, repo already initialized). Goal is to demonstrate domain expertise in electronics, photonics, antenna simulation, RF/signal processing through interactive browser-based demos — not just a static CV.

**Why:** Showcase capabilities to external audiences. Site should feel like a serious technical page, not a generic portfolio template.

**How to apply:** Keep everything client-side (no backend needed for v1). Precompute heavy simulation data locally in Python, export as JSON, build browser-based interactive visualizations on top. Stack discussion: Astro + React islands + Tailwind + Plotly.js + KaTeX is a strong candidate. Plain HTML/JS + Plotly + KaTeX is also viable if simpler is better.
